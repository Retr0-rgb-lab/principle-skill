# 货币危机信号扫描器(currency-crisis-scanner)

## R — 原文

引自《原则:应对变化中的世界秩序》第 35 章"货币、信贷、债务和经济活动的大周期":

> 央行印钞将导致货币贬值,使金融财富价值相对于实物财富价值下跌,直到金融资产的实际价值(经通胀调整后)低于实物资产的水平。这时,周期重新开始。

> 在第三类货币体系中,政府可以自由创造货币和信贷,只要人们对货币还有信心,这种体系就一直运营;一旦人们对货币失去信心,这种体系就会失灵。

> 由于货币贬值,人们逃离货币和以该货币计价的债务,转而选择其他货币和资产。

> 债务高企,利率无法充分下调,货币和信贷创造对金融资产价格的提振超过对实体经济活力的提振。

## I — 方法论骨架

货币危机不会一夜发生,是 6 个信号**逐个亮起**的过程。扫描时**必须同时看 6 个**,只看一个会误判:

1. **利率见顶信号**:名义利率已逼近零下限,降息空间用尽;央行开始用 QE / 负利率 / YCC 等非常规工具替代。
2. **印钞速度信号**:M2 同比增速连续 2 季度超过名义 GDP 增速 2 倍以上;央行资产负债表占 GDP 比重飙升。
3. **金融vs实物价差**:股票 / 债券 / 房地产总市值相对实体经济(收入、利润、产出)创历史新高,价格脱离现金流。
4. **信心裂缝信号**:黑市汇率溢价扩大、本币远期贬值预期升高、外汇储备连续环比下降、央行黄金储备增加。
5. **实物资产领涨**:黄金、房地产、大宗商品、必需品价格快速上涨,而股票 / 债券横盘——资金已开始从金融资产"逃向"实物。
6. **国债内循环**:本国央行持有本国国债比例超过 30%、海外央行连续净抛售、收益率曲线异常扭曲——主权信用进入内循环。

**预警分级**:6 个全绿 = 无 / 1-2 黄 = 早期 / 3-4 黄 = 中期 / 5+ 黄或 1 红 = 晚期 / 2 红及以上 = 临界(已开始贬值螺旋,需立即行动)。

## A1 — 书中的应用

### 案例 1 · 2020 后美联储无限 QE + 全球央行扩表(当前实时案例)
- **问题**:美国主导的货币体系现在处于 6 信号扫描的什么阶段?
- **方法论使用**:
  - 信号 1:2020 利率跌至零下限,美联储重启 QE、动用 YCC 工具 → 黄。
  - 信号 2:M2 同比增速 2020 单年达 25%,远超 GDP 增速 → 红。
  - 信号 3:美股市值 / GDP 创历史新高,房价 / 收入比超 2008 水平 → 黄。
  - 信号 4:新兴市场外储流失、央行集体增持黄金、本币远期贴水扩大 → 黄。
  - 信号 5:2022 起黄金、房地产、必需品领涨,股票横盘 → 局部黄。
  - 信号 6:Fed 持有美债占比超 20%,海外央行连续净抛售 → 黄。
- **结论**:综合评分**早期→中期过渡**(3-4 个黄灯),尚未到临界,但已脱离"无信号"区。
- **结果**:达利欧 2019 起建议配置 5–15% 黄金/通胀挂钩资产,2022 起公开预警"印钞导致的货币贬值正在发生"——本 skill 框架印证其信号判断。

### 案例 2 · 魏玛共和国 1921–1923(书中货币体系演化案例)
- **问题**:为何魏玛马克崩盘前 6 信号已全部亮起,但多数人没察觉?
- **方法论使用**:
  - 信号 1:魏玛央行贴现率虽未到零下限,但战债已无法靠利率工具压低 → 黄。
  - 信号 2:为支付赔款,央行直接为财政赤字印钞,基础货币月增 30%+ → **红**。
  - 信号 3:股市、债市相对实体经济严重虚高 → 红。
  - 信号 4:民众排队抢外汇、抢黄金、黑市汇率远超官方 → 红。
  - 信号 5:面包、煤炭、房租暴涨,金融资产无人问津 → 红。
  - 信号 6:德意志帝国银行被迫大量持有政府短期票据,海外资本清仓 → 红。
- **结论**:6 信号**全红——临界级**,"逃离货币→抛债务→抢硬资产"螺旋正式启动。
- **结果**:魏玛马克归零,经济自发回归"第一类货币"(黄金、外币、实物),印证书中"人们对货币失去信心,体系就会失灵"。

## A2 — 触发场景 ★

**适用场景**(6+ 个):
1. **看到 M2 飙升新闻**:判断"印钞速度"信号是否进入黄/红。
2. **跨国资产配置**:扫描美元 / 瑞郎 / 黄金 / 人民币 / 比特币的"危机安全度",决定避险比例。
3. **对冲基金经理日常扫描**:每个周末更新 6 信号打分,跟踪综合危机指数。
4. **央行研究员预警**:为内部报告输出"距离临界还差几个信号"的量化判断。
5. **新兴市场持仓评估**:扫描土耳其、阿根廷、埃及等是否已进入晚期/临界。
6. **滞胀时期市场分析**:1970s 复盘、2022 类比,判断当前是"早期还是晚期"。
7. **个人财富防御**:判断自己所在国家是否需要将部分资产转硬通货 / 海外 / 黄金。

**典型措辞**(5+ 个):
- "货币贬值螺旋开始了吗"
- "有没有资本外逃"
- "央行印钞到什么程度算危险"
- "会不会像 2008 / 魏玛 / 民国法币"
- "通胀要失控了吗"
- "现在该抢硬资产(黄金 / 房产 / 大宗)吗"
- "美元 / 人民币 / 日元还安全吗"

**与相邻 skill 的边界**:
- **vs `currency-regime-classifier`(货币体系分类器)**:本 skill 看**实时信号强度与时间窗口**(预警),后者看**体系结构属性**(诊断)。先分类,再扫描信号。
- **vs `debt-risk-triple-ratio`(债务风险三维比率)**:本 skill 是**宏观 6 信号扫描**(国家或全球级别),后者是**单笔 / 单主体三维打分**(国家/企业/个人债务)。
- **vs `purchasing-power-defender`(购买力防御)**:本 skill 输出**预警分级**(什么时候动),后者输出**行动方案**(怎么动)。先预警,再防御。

## E — 可执行步骤

1. **收集 6 类原始数据**(每类至少 1 个量化值):
   - 利率水平 + 央行资产负债表 / GDP;M2 同比 vs 名义 GDP;股市 / 房市市值比;外储变化、远期汇率、央行黄金增量;实物资产价格指数 vs 金融资产;央行持本国国债占比 + 外国央行净持仓。

2. **逐信号打分(0-100)**:每个信号独立打分,>60 黄、>80 红;不互相抵消。

3. **计算综合危机指数**:6 信号加权平均(权重可调,但金融vs实物价差与信心裂缝权重应稍高);输出 0-100 综合分。

4. **与历史案例比对**:把当前得分映射到魏玛(95)、2008 美国(60)、1997 亚洲(80)、2022 美国(50)、2024 美国(55)等历史样本。

5. **输出预警等级 + 关键转折点**:
   - **等级标签**:无 / 早期 / 中期 / 晚期 / 临界。
   - **最关键转折点**:哪个信号是下一个可能"由黄转红"的触发条件(如"再降息一次就触底""外储跌破 6 个月进口覆盖")。
   - **资产配置建议方向**:增持 / 减持 / 持有 + 具体品类(黄金、外币、海外房产、抗通胀债券、硬通货计价股票)。

6. **数据引用**:任何输出的数据点(数字、统计、事实)必须附带来源(URL 或数据库名 + 快照日期),无来源 → 不输出该数据点。例:`中国基尼系数 ≈ 0.47 (西南财大 CHFS 2018, https://...)`。

## B — 边界 ★

- **引用规则**:输出的所有数字、统计、事实性陈述必须附带可溯源出处(URL + 日期)。无法溯源的数据(民间说法、印象、听说)不得作为结论性陈述。

**反场景**(不要硬跑此 skill):
1. **单一指标异常**(CPI 略高但 6 信号其他都稳)——单点抖动不构成危机。
2. **短期做空 / 做多择时**——6 信号是结构性指标,不看周内动量。
3. **加密货币崩盘预测**——比特币等不在主权央行 6 信号框架内。
4. **完全不透明国家**——数据缺失无法完成 6 维扫描。
5. **公司债 / 个股风险**——6 信号是宏观维度,微观风险用 `debt-risk-triple-ratio`。

**常见失败模式**:
1. **只盯一个信号就下结论**——6 信号需同时看;只看 CPI 容易忽视"放水不进实体"的中期信号。
2. **信号亮但离临界还很远就开始恐慌**——早期/中期是"建仓硬资产"时机,不是"逃离一切"时机。
3. **忽视政策非线性反应**——政府可能突然控盘(资本管制、外汇干预、价格管制),框架无法预测突发政策。
4. **把"央行购金"等同"危机临近"**——央行购金是长期趋势,不必然指向短期危机。
5. **混淆"金融资产涨"和"实体经济好"**——这正是金融vs实物价差信号的核心,务必分别打分。

**作者盲点**:
- 达利欧的"信心崩塌"判断偏定性,6 信号本身需要**复合赋值**——单一定量阈值会失真。
- 对央行**非常规政策**(负利率、YCC、资本管制、数字货币 CBDC)的反应不敏感,需手动加入第 7 信号:"政策工具创新速度"。
- 对**小型开放经济体**与**大型储备货币国**未区分权重——同样 1 个红信号,对美元可能是早期,对土耳其可能是临界。
- 框架基于**主权法币 + 银行信贷**,对**加密资产、稳定币、影子银行嵌套**的"印钞替代效应"覆盖不足。

**邻近方法论区分**:
- **宏观对冲基金 risk parity 模型**(桥水 All Weather):看**资产配置风险预算**,与本 skill 互补——risk parity 决定"配多少",本 skill 决定"该不该配硬资产"。
- **BIS 早期预警指标**(Credit-to-GDP Gap、Debt Service Ratio):聚焦**信贷过热**,是本 skill 第 1-2 信号的细化,但不覆盖信心与实物价格信号。
- **Reinhart–Rogoff 违约信号体系**:基于**历史违约数据**,偏事后归纳;本 skill 是**前瞻扫描**,可早于违约事件 1-3 年预警。
- **Big Mac 指数 / PPP**:衡量**汇率偏离**,是第 4 信号(信心裂缝)的弱代理,远不如远期汇率 + 央行黄金增量直接。

## 数据源指引 (Data Sources)

本 skill 6 信号扫描必须用实时数据,每信号对应特定数据源:

- **FRED 货币与利率系列**: https://fred.stlouisfed.org/series/M2SL — 印钞速度 (信号 2), GFDEGDQ188S — 联邦债务/GDP (信号 1), DGS10 / TIPS 实际利率 (信号 1)
- **NBS 中国货币供应**: https://data.stats.gov.cn/ — 中国 M2 (信号 2)
- **BIS 实际有效汇率**: https://www.bis.org/statistics/ — 货币信心 (信号 4)
- **BIS 信贷/GDP Gap**: https://www.bis.org/statistics/ — 信贷泡沫信号 (信号 1)
- **Yahoo Finance 黄金/大宗**: https://finance.yahoo.com/ — 实物资产价格 (信号 5)
- **FRED 房地产价格指数**: https://fred.stlouisfed.org/series/CSUSHPISA — 实物资产信号 (信号 5)
- **IMF 资本流动数据**: https://data.imf.org/ — 资本外逃信号 (信号 4)
- **World Bank 黄金溢价数据**: https://www.gold.org/goldhub/data — 黑市汇率代理 (信号 4)

**优先级**(按信号):
1. FRED M2SL + GFDEGDQ188S (实时监控基础)
2. BIS REER + 信贷 Gap (中期预警)
3. Yahoo Finance 黄金 (市场情绪)
