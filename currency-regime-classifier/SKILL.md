# 货币体系分类器(currency-regime-classifier)

## R — 原文

引自《原则:应对变化中的世界秩序》第 3 章"货币、信贷、债务和经济活动的大周期":

> 几千年来,世界上一直存在三种类型的货币体系。第一类:硬通货(如金属硬币)。第二类:纸币(硬通货债权)。第三类:法定货币。

> 除非作为货币的金属或其他具有内在价值的大宗商品的供给增加,否则货币无法创造。

> 在第三类货币体系中,政府可以自由创造货币和信贷,只要人们对货币还有信心,这种体系就一直运营;一旦人们对货币失去信心,这种体系就会失灵。

引自第 4 章"货币价值的演变":

> 当央行发生这种情况时,它可以选择违约,或者与硬通货脱钩,发行货币,让货币贬值。

> 在新的体系中,美元与黄金挂钩,其他货币与美元挂钩。……要求兑换成黄金的票据数量远远超过可供兑换的黄金数量,从而导致美元与黄金脱钩。这催生了以美元为基础的法定货币体系。

## I — 方法论骨架

达利欧把几千年的货币史压缩为**三类**循环演进的体系:

1. **类型 1 · 硬通货**:金属币或实物货币,内在价值 = 购买力;货币数量受矿产/实物供给限制。
2. **类型 2 · 纸币(硬通货债权)**:纸币代表对硬通货的可兑换索取权,需锚定黄金或硬资产;央行不能自由创造。
3. **类型 3 · 法定货币**:无锚定的政府信用货币,政府可自由创造货币与信贷;**体系的稳定完全依赖"信心"而非锚**。

**演进路径**:1 → 2 → 3 →(信心崩溃)→ 1,周而复始。三类切换由两道"门"决定:

- **门 A(1→2)**:是否存在可兑换的硬通货锚定。一旦央行印钞超过储备支撑,即被迫"与硬通货脱钩"。
- **门 B(2→3)**:央行是否已获得**单方面货币发行权**,无需外部硬通货兜底。

**核心判断原则**:不要用"汇率稳定""通胀率"替代"信心";法定货币体系下,只要持有者仍愿意以本币储蓄、计价、计价债务,体系就还在运转。

## A1 — 书中的应用

### 案例 1 · 美元体系演变(1944→1971→当下)

- **问题**:美元经历了哪几次"门"的穿越?现在是哪种类型?
- **方法论使用**:
  - 1944 布雷顿森林:35 美元兑 1 盎司黄金,各国外储以美元计,**类型 2(纸币锚定黄金)**——门 A 闭合。
  - 1971 尼克松脱钩:法国等央行用美元挤兑黄金,储备不足以兑付,美国单方面关闭黄金窗口,**门 A 打开,从 2 直接跃迁到 3**。
  - 当下:美联储可无限创造美元、美国国债以本币计价,**类型 3 完全体**。
- **结论**:达利欧把 1944–1971 视为"门 A 关闭期",1971 之后为"门 B 完全打开、纯靠美国国力+军事+石油美元维持信心"的类型 3。
- **结果**:判断未来 5–10 年美元去向的钥匙是**"信心"指标**(外国央行美债持仓、美元在国际储备中的占比、黄金储备环比增长),而非通胀本身。

### 案例 2 · 魏玛德国马克(1921–1923 信心崩溃)

- **问题**:为何一战战败的德国马克从"稳如国债"一夜归零?
- **方法论使用**:
  - 起点:魏马克诞生时规定锚定黄金(类型 2)。
  - 过程:战争赔款 → 央行直接为财政赤字印钞 → **门 A 实质性脱钩** → 演变为类型 3。
  - 阈值:印刷马克的速度追不上汇率崩盘,民众抛马克抢黄金、美元、商品。
  - 终局:纸马克一文不值,经济自发**回归类型 1**(黄金、外币、实物交易)。
- **结论**:印证书中"人们对货币失去信心,体系就会失灵"——魏玛不是被通胀打败,而是被**对通胀的预期**打败,触发资金逃离。
- **结果**:回归到"以物易物 + 硬通货双轨",这是达利欧强调类型 3 循环回归类型 1 的活样本。

### 案例 3 · 民国晚期金圆券(1937–1949,中国历史周期)

- **问题**:法币→金圆券→崩溃,为何"金本位背书"也救不了?
- **方法论使用**:金圆券发行公告虽写明黄金准备,但**实际无足额硬通货储备**,属"假门 A 关闭、真门 B 全开";一旦民众挤兑黄金,发行方无力兑付,**信心立刻清零**。
- **结论**:门 A 是否真正关闭,**看"准备金真实性"**而非"政策文本措辞"。文字上的金本位不等于事实上的类型 2。

## A2 — 触发场景 ★

**典型触发措辞**(5+ 个):
- "美元 / 人民币 / 日元 / 欧元是什么货币体系?"
- "这会不会像魏玛 / 金圆券 / 民国法币?"
- "数字货币 / 央行数字货币 / 比特币属于第几类?"
- "央行要不要与黄金脱钩 / 重新挂钩?"
- "货币信心还有吗?什么指标看信心?"
- "去美元化是好事还是坏事?"
- "美元崩溃后会回归黄金吗?"

**适用场景**(5+ 个):
1. **跨国资产配置**:评估美元 / 瑞郎 / 人民币 / 日元 / 黄金的"体系属性",决定配置比例。
2. **新兴市场投资**:判断某国货币还停留在类型 2 还是已彻底进入类型 3,评估贬值路径。
3. **央行数字货币(CBDC)分类**:判定某国 CBDC 落在三类哪一格(延续锚定 vs 完全法定)。
4. **历史货币制度比较**:把布雷顿森林之前的英镑、清末银两、魏玛马克、当下美元并排归类。
5. **新闻解读**:对"去美元化""金砖货币""金本位回归"等热门议题做框架化归类。
6. **识别"伪锚定"骗局**:识别空有黄金背书文本、无真实储备的"假类型 2"。

**与相邻 skill 的边界**:
- **vs `currency-crisis-scanner`(货币危机扫描器)**:本 skill 输出"体系类型 + 转换方向"(诊断),后者输出"危机信号强度 + 时间窗口"(预警)。先分类,再扫描。
- **vs `debt-risk-triple-ratio`(债务风险三维比率)**:本 skill 看**货币体系结构**,后者看**具体债务主体的三维比率**。体系分类决定"环境",三维比率决定"个体"。
- **vs `cycle-stage-detector`(周期阶段定位器)**:本 skill 是**货币维度专项**(只回答"货币在哪一格"),后者是**宏观多维度定位**(政治、经济、军事一起定位)。

## E — 可执行步骤

1. **收集三类信号**(完成标准:每一类至少有 1 个可观察事实):
   - 类型 1 信号:央行黄金储备吨数、是否允许黄金自由交易、是否承认商品货币。
   - 类型 2 信号:是否存在法定可兑换的硬通货锚定、是否设有固定平价、储备/货币供应覆盖率。
   - 类型 3 信号:央行独立性、是否使用本币为唯一法定清偿、是否常态化货币创造。

2. **判断门 A(1→2)**:是否有真实可兑付的硬通货锚?若答案为"是",停留在类型 1→2 区间;若答案为"否但曾有过",直接认定已跨过门 A。

3. **判断门 B(2→3)**:央行是否已获得**事实上的**单方面货币发行权?是否仍受外部硬通货约束?若否,已进入类型 3。

4. **评估货币信心(类型 3 内部)**——这是最关键一步,看四个观察点:
   - 外国央行储备币种占比变化
   - 国内居民储蓄/资产是否仍以本币计价
   - 本币计价国债是否仍能以正实际利率发行
   - 是否出现囤金、外币、抢购实物商品的迹象

5. **输出"货币体系诊断卡"**(固定模板):
   - 体系类型标签(1/2/3/混合型)
   - 门 A 状态(开/关/半开)
   - 门 B 状态(开/关)
   - 信心等级(高/中/低 + 1 句依据)
   - 转换预警(下一阶段最可能事件 3 条内)

## B — 边界 ★

**反场景**(不要硬跑此 skill):
1. **加密货币估值**——比特币、以太坊、稳定币不在"主权央行发行"框架内,需用其他框架。
2. **短期汇率预测**——本 skill 看"结构"不看"动量",不适合"下个月美元会不会涨"。
3. **个人换汇决策**——旅行换汇、留学换汇属个体决策,不适合宏观体系分类。
4. **单国货币历史叙事**——只讲故事不归类,本 skill 必须给出当前类型标签。
5. **完全不透明国家**——无法获取外储、债务、央行数据的小国或战乱国家,无法走完五步。

**常见失败模式**:
1. **把"通胀率"当信心指标**——通胀是结果,信心才是原因;量化信心没有"标准刻度",需用储备占比、储蓄币种、抢购实物等多源信号交叉验证。
2. **严格三分类**——现实中多为**混合体系**(如香港的联系汇率仍在门 A 关闭、门 B 也部分打开),不要硬塞进单一类型。
3. **把"汇率稳定"等同"货币信心强"**——管制下的汇率稳定可掩盖真实信心流失,须看外储变化。
4. **把"金本位文本"等同"金本位事实"**——金圆券、伪金本位的失败证明文字背书不算数。
5. **只看本国不看外部**——类型 3 的信心很大程度由"外部央行/储户是否还持有本币"决定,必须看外部筹码。

**作者盲点**:
- 框架以**主权央行 + 银行信贷**为基础,对**加密货币、链上稳定币、超主权货币(IMF SDR、欧元)**划分模糊,需要扩展。
- 对**小型开放经济体**的"信心"判断依赖外部环境,易被大国货币政策溢出效应主导,需谨慎归因。
- **数字货币 / CBDC**:CBDC 大概率延续类型 3(央行仍垄断发行),但其**可编程性**可能改变"信心机制"——这是达利欧时代未预料的变量。

**邻近方法论区分**:
- **Friedman 货币主义(数量论 MV=PY)**:关注货币数量与价格关系;本 skill 关注**货币属性分类**,两者互补,数量论是"火候"、本框架是"锅型"。
- **Hayek 货币非国家化**:主张私人发行竞争货币对抗国家垄断;本 skill 不讨论"应否垄断",只描述现状分类。
- **Keynes 货币理论**:聚焦利率与流动性偏好;与本 skill 是**正交维度**,Keynes 看需求侧,本 skill 看供给侧制度。

## 数据源指引 (Data Sources)

本 skill 判定货币体系类型必须用实时数据,包括硬通货锚定、央行发行安排、货币信心信号:

- **IMF IFS (国际储备)**: https://data.imf.org/ — 各国黄金/外汇储备
- **World Gold Council 央行持金**: https://www.gold.org/goldhub/data — 央行黄金持有量
- **BIS 央行资产负债表**: https://www.bis.org/statistics/ — 货币发行机制
- **IMF COFER**: https://data.imf.org/ — 储备货币构成(美元/欧元/人民币占比)
- **FRED M2 / 实际利率**: https://fred.stlouisfed.org/series/M2SL — 货币供给与利率
- **NBS 货币供应**: https://data.stats.gov.cn/ — 中国 M0/M1/M2

**优先级**(按判断步骤):
1. IMF COFER + BIS 央行资产负债表(体系分类核心)
2. World Gold Council(锚定判据)
3. FRED/NBS 货币供应(信心指标)
