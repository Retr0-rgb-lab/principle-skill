# cycle-correlation-scenario · 三周期交叉推演器

## R — 原文

**第 34 章 · 2 决定因素**(因果链 + 有限可能性)

> 演变是缘于符合逻辑的因果关系而发生的。现有情况和决定因素驱动着各种变化,而这些变化又创造出一套新的情况和决定因素,从而推动新的变化,因为一组特定情况创造出一组有限的可能性。

**第 33 章 · 1 大周期简述**(顶部五大特征)

> 顶部阶段以各种形式的过度状况为特征:(a)高负债,(b)财富、价值观和政治差距巨大,(c)教育水平和基础设施不断下滑,(d)国家内部不同群体之间发生冲突,(e)过度扩张的国家受到新兴对手的挑战,引发国家之间的争斗。

**第 38 章 · 6 外部秩序和混乱大周期**(国际秩序本质)

> 国际秩序与其说是遵循国际法,不如说是丛林法则。所有最伟大的国家都在发展世界领先的金融中心,从而吸引和分配当时的资本。

**第 37 章 · 5 内部秩序和混乱大周期**(分散效应 + 5 阶段量化)

> 各国间的区别产生了一种分散效应,这使全球的演进过程比任何单个国家都更平稳。我们可以把民粹主义和两极分化作为指标——民粹主义现象越多,两极分化越严重,一个国家在内部周期的第五阶段就会走得越远。

## I — 方法论骨架

**核心命题**:世界不是三条独立曲线,而是一个 3×3 的因果耦合系统。**货币-债务周期**(央行印钞/去杠杆) ↔ **内部秩序周期**(6 阶段从崛起到革命) ↔ **外部秩序周期**(霸权崛起/衰落/战争) 互相喂养、彼此放大。

**三组耦合机制**:
- **债务 → 内部**:央行印钞贬值货币 → 金融资产跑赢实物 → 贫富差距扩大 → 民粹抬头 → 第 5 阶段加速。
- **内部 → 外部**:内部第 5 阶段冲突消耗国力 → 国库被民粹补贴掏空 → 国际让位给新兴对手 → 储备货币受质疑。
- **外部 → 债务**:国际竞争迫使军费/产业政策加码 → 财政恶化 → 国内债务压力上升 → 央行再次印钞 → 新一轮贬值。

**3×3 矩阵**:9 格中至少 4 格代表"三周期均处良性区"——这是牛市情景;最右下角"债务崩盘 + 内部第 5 阶段 + 外部战争"为极端情景。**矩阵本身不输出答案**,只用来穷举可能性。

**4 情景生成法(剪枝是质量底线)**:9 格理论组合 + 2 维时间分支 = 54 种理论可能。**严格剪枝到 4-6 个**:从"当前已确定位置"出发,只保留概率 ≥5% 且因果链自洽的分支。其余视为不可能事件,直接丢弃。

**4 个基线情景**(权重由当前数据校准,非平均分配):
1. **牛市**(各周期彼此缓冲,温和正循环): 15-25%
2. **基准**(延续当前趋势,缓慢恶化或修复): 30-50%
3. **熊市**(一个周期拖垮另一个,级联启动): 20-30%
4. **极端**(三周期同时恶化至终局,需三条件齐发): 5-15%

**有限可能性原则**:不要给用户列出"54 种可能";只告诉他"4-6 种可能性 + 概率",这是达利欧框架的方法论硬约束。

## A1 — 书中的应用

**案例 1 · 1930s 大萧条(极端情景教科书)**
- **问题**:1914-1945 这段为什么是"最剧烈"的世界动荡?
- **方法论使用**:三周期同时处于终局区——(a)债务周期顶部:一战债务货币化后,1929 信贷泡沫破裂;(b)内部第 5 阶段:德国魏玛恶性通胀、苏联集体化、美国大萧条;(c)外部秩序:凡尔赛体系无法约束轴心国扩张,走向二战。
- **结论**:达利欧判断这是"500 年来三大秩序同时崩溃"的典型,印证"三周期共振 = 极端情景"。
- **结果**:从 1914 到 1945 年,全球 GDP 在战争与萧条中损耗 30%+;以 1933 布雷顿森林重建为终,新秩序诞生。

**案例 2 · 1971 布雷顿森林倒塌(单点让位触发三周期重构)**
- **问题**:美元为何在 1971 突然与黄金脱钩?
- **方法论使用**:货币周期单点让位(美国黄金储备无法支撑对外负债) → 内部秩序恶化(越战开支 + 福利扩张) → 外部秩序重建(美元贬值但仍维持储备地位,因无替代)——三周期重构的"基准情景"模板。
- **结论**:单一周期顶部的让位,带动其他两周期重新定位,但未触发极端,因无强对手(苏联尚未市场化)。
- **结果**:1971-1980 滞胀、美元-黄金脱钩,但美股 1980s 启动长牛;证伪"美元崩溃 = 美国崩溃"的线性外推。

**案例 3 · 19 世纪末英国 → 20 世纪初美国(霸权交接的基准情景)**
- **问题**:英美霸权交接为何相对平稳,未爆发全面战争?
- **方法论使用**:英国在第 5 阶段(1870 后殖民成本超负荷)、美国在第 2 阶段(重建期)——**两国处于不同内部阶段产生"分散效应"**,让位于英美合作而非英美对抗;债务周期方面英国国债/GDP 高但市场仍信,美国工业崛起但货币尚弱。
- **结论**:分散效应是基准情景能保持温和的根本原因——不是"美国变强",而是"英国仍强 + 美国未强到挑战底线"。
- **结果**:1890-1914 帕克伯恩协约体系;**但分散效应耗尽后(1914)即触发极端情景**——这正是为什么 1914-1945 是"两段拼接",而不是两个独立大周期。

## A2 — 触发场景

**适用场景(6 个)**:
1. **家族办公室 10 年规划**:评估"未来 10 年最坏会变怎样",决定资产跨币种、跨地域配置比例。
2. **家族跨代财富传承**:第三代以后是否仍能保住购买力——必须考虑三周期叠加对制度/货币的冲击。
3. **国家战略部评估**:外交部 / 智库评估未来 5-10 年国际秩序走向,决定对外政策与军事投入。
4. **外资机构战略部**:跨国机构评估某新兴市场国家"未来 5 年还能不能投资",需看三周期耦合而不是单看 GDP。
5. **跨国企业避险**:产业链是否要"中国 + 1"或"美国 + 1",取决于三周期是否同时走向终局。
6. **个人面临职业转型抉择**:长期主义决策(读博、移民、跨行业)需对"未来 10-20 年世界秩序"有判断。

**典型措辞(5+)**:
- "三个周期叠加会怎样?"
- "最坏情况下 X 国会变怎样?"
- "未来 10 年推演一下。"
- "危机会不会同时爆发?"
- "帮我看看尾部风险。"
- "现在该 all-in 还是 hedge?"
- "美元 + 美国内政 + 大国竞争,哪一项会先爆?"

**与相邻 skill 的区分**:
- **vs `cycle-stage-detector`**:那是单一坐标定位(答"在哪"),本 skill 是三周期耦合情景树(答"会怎样");先定位、再交叉。
- **vs `worst-case-decision-guard`**:那是微观决策框架(答"我该怎么办"),本 skill 是宏观推演工具(答"世界会怎样"),输出物是情景树不是行动清单。
- **vs `currency-crisis-scanner`**:那是单一信号扫描(答"货币危机信号亮了几盏"),本 skill 是综合推演器,把货币信号作为输入之一。

## E — 可执行步骤

1. **收集三个周期的当前阶段**(可调 `cycle-stage-detector`)。完成标准:三周期各有明确阶段标签(债务:扩张/顶部/通缩去杠杆/再通胀去杠杆;内部:1-6 阶段;外部:霸权上升/主导/衰退/竞争/重构)。
2. **评估 3×3 耦合矩阵**。完成标准:9 格中至少识别出 4-5 格"因果自洽",其余格标注为"低概率"或"不可能"。每格写出"为什么走 / 为什么不走"。
3. **严格剪枝到 4-6 情景**。完成标准:从 9 格中挑出概率 ≥5% 的 4-6 条路径,每条对应一个情景标签(牛市 / 基准 / 熊市 / 极端);剩余全部丢弃,不写进最终输出。**这条是质量底线**——列出 10+ 情景 = 完全失焦。
4. **为每情景分配概率权重**。完成标准:四档概率合计 = 100%;每档配 1-2 句"为什么是这个权重"(例如"基准给 35% 因为当前债务周期刚开始去杠杆,内部极化但未到第 5 阶段")。
5. **输出"情景树 + 历史案例对照 + 行动建议"**。完成标准:四档情景 × 三列(触发路径 / 关键阈值 / 历史案例),末尾 3-5 条"跨情景通用行动"(例如"实物资产 + 跨币种 + 保留退出路径")。
6. **数据引用**:任何输出的数据点(数字、统计、事实)必须附带来源(URL 或数据库名 + 快照日期),无来源 → 不输出该数据点。例:`中国基尼系数 ≈ 0.47 (西南财大 CHFS 2018, https://...)`。

## B — 边界

**反场景(不应触发)**:
- 短期交易(1-6 个月):周期不可识别,本 skill 失焦。
- 单一周期分析(只问债务 / 只问内部秩序):应拆解为 `cycle-stage-detector` 或 `currency-crisis-scanner`。
- 用户拒绝接受"概率"概念(只要确定答案):本 skill 的核心就是概率权重。
- 数据高度不透明国家(战乱中、统计失真):无法做基线校准。
- 全新范式场景(纯加密经济体、AI 主导、去国家化):框架基于历史大周期,外推有限。

**常见失败模式**:
1. **列出 10+ 情景 = 完全失焦**——把 54 种理论组合全摆出来,等于告诉用户"什么都有可能",反而无信息。
2. **把"中性"情景当成"无变化"**——基准情景也是"演变"而非"静止",要描述方向而非"什么都不会发生"。
3. **概率权重凭感觉**——每档必须有 1-2 句依据,否则就是占星术。
4. **忽视"分散效应"**——只看单一国家会崩溃,忽略全球 200 国处于不同阶段这一关键缓冲器。
5. **把"尾部风险"等同于"极端情景"**——熊市(20-30%)与极端(5-15%)是不同等级,前者更值得关注,因为概率足够高。

**作者盲点 / 时代局限**:
- 框架基于"已有大周期历史"的经验,但**现代化技术变量(AI、加密、地缘技术战)的权重可能不足**——达利欧本人也承认"技术奇点"在他的驱动因素列表外。
- 情景依赖当前数据,5+ 年推演的精度有限——更长时间必须用"演变逻辑"而非"线性外推"。
- **对小国/封闭经济体适用性差**——框架以大国大周期为基础,小国往往是"被波及方"而非"主角"。

**邻近方法论的区分**:
- **蒙特卡洛模拟**:数学概率分布,假设独立同分布;本 skill 用因果耦合,不假设独立性。
- **RAND 公司战略推演**:军事情景,聚焦对手行为建模;本 skill 更宽,含经济 + 政治 + 国际秩序。
- **黑天鹅理论(Nassim Taleb)**:强调"不可预测的极端事件",本 skill 反而主张"在已知因果机制下极端可被推演",两者相反。
- **引用规则**:输出的所有数字、统计、事实性陈述必须附带可溯源出处(URL + 日期)。无法溯源的数据(民间说法、印象、听说)不得作为结论性陈述。
