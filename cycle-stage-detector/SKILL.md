# cycle-stage-detector · 周期阶段检测器

## R — 原文

**第 33 章 · 1 大周期简述**(顶部五大特征,中文版第 57 行附近原文)

> 顶部阶段以各种形式的过度状况为特征:(a)高负债,(b)财富、价值观和政治差距巨大,(c)教育水平和基础设施不断下滑,(d)国家内部不同群体之间发生冲突,(e)过度扩张的国家受到新兴对手的挑战,引发国家之间的争斗。

**第 31 章 · 引言**(螺旋上行与单循环,原书开篇)

> 进化是走向进步的上行轨迹,而进步源于适应和学习。周期围绕着上行轨迹而反复出现。这条轨迹就像一个上行的螺丝锥……进化和周期共同构成螺旋式上行轨迹,这条上行轨迹显现在一切事物(财富、政治、生物、技术、社会学和哲学等)中。

**第 37 章 · 5 内部秩序和混乱大周期**(第 5 阶段量化指标)

> 我们可以把民粹主义和两极分化作为指标。民粹主义现象越多,两极分化越严重,一个国家在内部周期的第五阶段就会走得越远,越接近内战和革命。在第五阶段,中间派变成少数派。在第六阶段,中间派不再存在。

**第 44 章 · 11 美国和美元的大周期兴衰**(美国当前定位)

> 美国在自身大周期中约处于 70% 的位置……基本上讲,借款实际上是免费的,因此从事投资和企业经营的借款者利用这些廉价融资进行购买,从而拉动股价和公司利润增长。这些资金并未按比例流向不同阶层,因此贫富差距和收入差距继续扩大。与此同时,政治鸿沟加大,一边是日益极端的共和党人,另一边是日益极端的民主党人。

## I — 方法论骨架

达利欧把国家的兴衰看作一段 100-300 年的"大周期",由两条正交曲线组成:一条是**内部秩序**(从"崛起→建设→和平繁荣→过度与泡沫→冲突与民粹→内战/革命"共 6 阶段),另一条是**长期债务周期**(从去杠杆→信贷扩张→顶部→去杠杆再爆发→新货币秩序)。

本 skill 只做"宏观定位",不做"具体预测":

1. **顶部信号灯**:五大顶部特征是高负债、价值观/政治鸿沟、教育基建下滑、内部冲突、外部对手崛起。点数越多,越接近第 4→5 阶段。
2. **第 5 阶段量化**:民粹主义抬头、两极分化加剧、中间派消失、媒体失真、未经历过苦日子的一代崛起。
3. **债务周期位置**:看央行是否已用尽降息空间、债务货币化是否主导、资产价格是否严重偏离实物财富。
4. **历史类比**:同一阶段的不同国家往往呈现相似的政治经济模式——这是"特定情况创造有限可能性"。
5. **分散效应**:各国处于不同阶段,世界的演进因此比单一国家更平滑。

## A1 — 书中的应用

**案例 1 · 荷兰帝国(17 世纪 → 18 世纪衰落)**
- **问题**:荷兰如何从世界金融霸主(阿姆斯特丹证券交易所、第一家股份制公司)走向衰落。
- **方法**:沿 6 阶段框架定位——荷兰经过第 2→3 阶段建立全球贸易与金融中心,18 世纪依次显现(a)债务高企(英法战争借款)、(b)财富差距扩大、(d)内部政治分裂;同时被英国通过军事+金融优势取代,落入第 4→5 阶段。
- **结果**:荷兰盾失去储备货币地位,18 世纪末已被英镑边缘化,印证达利欧"储备货币寿命长于国家实力"的论断。
- **结论**:高度依赖金融中心定位的国家,一旦内部秩序退化,衰退会非常快(从第 3→4 阶段到第 5 阶段,往往只有 1-2 代人)。

**案例 2 · 大英帝国(1945 后衰落,达利欧未详述但属于典型案例)**
- **问题**:英国如何在两次世界大战后从霸主变回普通岛国。
- **方法**:第 44 章用同一框架看美国,对英国可类推——二战军费债务化 + 殖民地独立运动(印度 1947)+ 美国崛起替代其货币地位,英国从第 3 阶段直接滑入第 5 阶段。
- **结果**:英镑 1944 年布雷顿森林会议被美元替代为储备货币核心,1949 年大幅贬值,印证"帝国失势先从货币开始"。
- **结论**:第 5 阶段的典型特征是"国际地位滑落先于内部秩序衰败"。

**案例 3 · 美国当前(达利欧 2020s 自我定位)**
- **问题**:美元大周期走到哪了?美国是否进入第 4-5 阶段?
- **方法**:达利欧直接判断"美国约处于 70% 位置";五大信号灯中,(a)联邦债务 / GDP 超 120%、(b)政治极化第 37 章量化指标、(d)中期难以调和、(e)中俄挑战——至少 4 盏已亮。
- **结果**:他主张进入"长期债务周期顶部"阶段,但距 6 阶段终点尚远,战争与内战概率仍低。
- **结论**:宏观定位不等于崩溃预言;关键是用定位指导"未来 5-10 年的资产/职业/移民布局"。

## A2 — 触发场景

**适用场景(5 个)**:
1. **国家维度**:用户问"美国/中国/日本/某新兴市场现在到周期哪了?""X 国是不是到顶了?""历史会不会重演?"
2. **行业维度**:"半导体 / 互联网 / 新能源现在像 1999 年的互联网吗?""地产 / 银行 / 制造业是不是进入顶部?"
3. **公司维度**:类似"苹果 / 字节 / 某巨头是不是开始走下坡路的帝国?"——可套用 6 阶段简化版。
4. **资产配置维度**:"当前应不应该继续持有美元资产 / 中国房产 / 美债?"
5. **历史类比**:"现在像 1929 / 1971 / 1989 / 2008 的哪一段?""什么是下一个清朝?"

**典型措辞(5+)**:
- "这是不是泡沫顶部?"
- "我们现在处在周期的哪个阶段?"
- "X 国像历史上的什么时候?"
- "帝国会不会衰落?衰落到什么程度?"
- "大周期会重演吗?"
- "美元 / 人民币还能信多久?"
- "这次和以前有什么不一样?"

**与相邻 skill 的区分**:
- **`debt-risk-triple-ratio`**:那是单点风险量化,产出"三个比率 + 是否违约",回答"风险有多大";本 skill 输出"宏观阶段定位 + 历史类比",回答"我们现在在哪、像谁"。
- **`worst-case-decision-guard`**:那是个人决策框架,产出"该怎么做";本 skill 是宏观诊断,产物只是定位图,不直接给出行动清单。

## E — 可执行步骤

1. **收集五大信号灯数据**(完成标准:每个信号灯都有数据/事件支撑,**任意 3 盏亮**就视为"进入第 4 阶段后期")。
   - (a) 总债务 / GDP、(b) 基尼系数 / 政治极化指数、(c) 教育支出、基建质量评分、(d) 内战 / 抗议频度、(e) 新兴对手 GDP 占比追赶速度。
2. **定位内部秩序阶段**(判停条件:**至少 4 个特征匹配**才能给出阶段标签,否则说"数据不足无法判断")。
3. **定位长期债务周期位置**(完成标准:给出"扩张期/顶部/通缩去杠杆/再通胀去杠杆"四选一,附 1-2 句依据)。
4. **匹配历史类比**(完成标准:从书中 500 年案例库(荷兰、英国、美国、中国王朝)选 1-2 个相似阶段,说明"像 + 不像")。
5. **输出"周期阶段诊断卡"**(模板固定):阶段标签 / 五盏灯 / 债务周期位 / 历史类比 / 下一阶段大概率事件(3 条内)。
6. **数据引用**:任何输出的数据点(数字、统计、事实)必须附带来源(URL 或数据库名 + 快照日期),无来源 → 不输出该数据点。例:`中国基尼系数 ≈ 0.47 (西南财大 CHFS 2018, https://...)`。

## B — 边界

**反场景(不应触发)**:
- 短期交易(日内 / 周内)询问、判断个股择时。
- 个人理财(怎么配置 100 万)。
- 用户只想听"会涨还是会跌",不要框架、不要历史。
- 国家与世隔绝 / 数据高度不透明(如战乱中的国家、无法获取 GDP 与政治数据的小经济体)。
- 加密货币 / 央行数字货币 / 全新货币范式(框架未覆盖)。

**常见失败模式**:
1. **只看 GDP,不看政治**:忽略民粹主义、极化、媒体失真——而这些正是第 5 阶段量化指标。
2. **用短期数据做长期周期判断**:拿 1-2 年经济波动判断 100 年大周期,颗粒度错配。
3. **线性外推"这次不一样"**:把和平繁荣期(第 3 阶段)直接外推到永远,忽略上升轨迹中的周期回调。
- **引用规则**:输出的所有数字、统计、事实性陈述必须附带可溯源出处(URL + 日期)。无法溯源的数据(民间说法、印象、听说)不得作为结论性陈述。

**作者盲点 / 时代局限**:
- 框架以**大型、开放、金融化程度高**的经济体为基础,对小型封闭经济体直接套用可能失真。
- 数据可获取性依赖发达统计体系,对**前现代或数据稀缺国家**需要补足定性观察。
- **加密资产 / 数字货币**:不在达利欧的"硬通货→纸币→法定货币"三阶段框架内,需扩展。
- **技术奇点 / AI 加速**:第 33 章的"基本面驱动力"列表未必充分纳入未来技术变量。

**邻近方法论的区分**:
- **康波周期(50-60 年长波,熊彼特)**:聚焦技术革命驱动的经济长波,周期长度与因果机制不同;本 skill 更宽,含政治 + 货币 + 国际秩序维度。
- **熊彼特创新周期**:聚焦"创造性破坏"对企业与行业的冲击,不直接处理国家兴衰;本 skill 是国家层面。
- **雅斯贝斯"轴心时代" / 汤因比"文明史"**:历史哲学框架,周期长度以"千年/世纪"计;本 skill 以"几十年到三百年"为单位,粒度更短、可操作。

## 数据源指引 (Data Sources)

本 skill 定位国家/经济体的"大周期阶段"必须依赖实时量化数据,仅靠内部知识极易判错(信号灯 a/b/e 都需权威数据源交叉验证)。调用时使用 WebFetch / WebSearch 从以下权威源按"5 大信号灯 + 债务周期位置"维度拉取最新数据:

- **World Bank Open Data**: https://data.worldbank.org/ — 跨国对比首选,关键指标: `NY.GDP.MKTP.KD.ZG` (GDP 增长率)、`GC.DOD.TOTL.GD.ZS` (政府债务/GDP)、`SI.POV.GINI` (基尼系数),覆盖信号灯 (a)(b)(e)
- **FRED (St. Louis Fed)**: https://fred.stlouisfed.org/ — 美国深度数据,关键 series: `GFDEGDQ188S` (联邦债务/GDP)、`GDPC1` (实际 GDP)、`CPIAUCSL` (CPI)、`DGS10` (10 年期国债收益率),用于信号灯 (a) 与债务周期位置
- **NBS 国家统计局**: https://data.stats.gov.cn/ — 中国宏观核心指标 (GDP / CPI / 失业率 / 财政收入 / 地方政府债),用于信号灯 (a)(c) 中国侧
- **IMF WEO**: https://data.imf.org/ — 全球经济展望与债务/GDP 历史对比,辅助信号灯 (a)(e) 跨国预测
- **V-Dem 民主指数**: https://www.v-dem.net/ — 第 5 阶段量化核心: 评估政治极化、媒体失真、司法独立,关键子项 v2x_polyarchy / v2xme_altinf,用于信号灯 (b)(d)
- **Polity V 政治体制评分**: https://www.systemicpeace.org/ — Polity score (-10 威权 ~ +10 民主) + 政变/内战事件流,量化内部秩序第 5→6 阶段
- **BIS Credit-to-GDP Gap**: https://www.bis.org/statistics/ — 长期债务周期定位关键指标 (信贷 / GDP gap + Debt service ratio),判停"扩张期 / 顶部 / 去杠杆"

**优先级**(从 5 盏信号灯 + 债务周期覆盖度排序):

1. **World Bank Open Data** — 跨国对比首选,一站覆盖 a/b/e 三盏灯 + 多国历史
2. **FRED** — 美国深度数据,信号灯 a 与债务周期位置的权威源
3. **V-Dem + Polity V** — 政治极化量化,第 5 阶段判断必备(单凭 GDP 必然漏判)

备用: NBS (中国侧指标)、IMF WEO (跨国预测)、BIS (债务周期定位)、Freedom House (政治自由度交叉验证)
