# 债务风险三维比率评估器(debt-risk-triple-ratio)

## R — 原文

引自《原则:应对变化中的世界秩序》第三部分"货币、信贷、债务和经济活动的大周期":

> 要想正确评估持有债务这颗定时炸弹的风险和回报,我们就必须时刻注意到需要偿还的债务与可供偿还的硬通货的比率、需要支付的债务金额与债务人的现金流的比率,以及投资者获得的利息回报。

> 投资者需要经常自问的一个重要问题是,你得到的利息金额是否足以弥补你所面临的货币贬值风险。

> 所有货币都会贬值甚至消亡。一旦出现这种情况,现金和债券(即获得货币的承诺)就会贬值甚至变得一文不值。…… 因此,黄金(其次是白银)既可以作为安全的交换媒介,也可以作为安全的财富贮藏手段。

> 通过债务融资来购买商品、服务和投资资产的情况迅速增多,因此债务增速超过偿付债务的未来现金流增速。这就产生了泡沫。

## I — 方法论骨架

把"一笔债"拆成三个**独立**比率,任何一维超标都构成结构性风险:

1. **硬通货覆盖率** = 黄金 + 外汇储备 + 高流动性实物资产 / 需偿还的外币计价债务
   - 衡量"兑付能力":能不能用真"硬资产"还债。
   - < 30% 红 / 30–80% 黄 / > 80% 绿。

2. **现金流负担率** = 年还本付息 / 年可支配现金流(经营性现金流或税后收入)
   - 衡量"持续偿付能力"。达利欧反复指出,债务增速一旦超过现金流增速就产生泡沫。
   - > 40% 红 / 20–40% 黄 / < 20% 绿。

3. **实际利率 vs 贬值风险** = 名义利率 − 预期本币贬值率
   - 衡量"持有债务是否被通胀反向收割"。负值意味着借钱的人受益、放钱的人被没收。
   - < −2% 红 / −2% ~ +1% 黄 / > +1% 绿。

**核心原则**:三维之间**不能互相补偿**。硬通货低但现金流强 → 危机时无储备兜底;现金流强但实际利率长期为负 → 财富被持续稀释;现金流差且贬值预期高 → 双重塌方。

## A1 — 书中的应用

### 案例 1:美国国债(2008–2024)
- **问题**:美国国债是否仍是"无风险资产"?
- **方法论使用**:
  - 硬通货覆盖率:外储 + 黄金 ≈ 5800 亿美元,对外计价债务超 30 万亿美元 → **红**(覆盖率 < 2%)。
  - 现金流负担率:年付息约 1 万亿美元,联邦财政收入约 4.4 万亿 → 约 23% → **黄**。
  - 实际利率 vs 贬值:M2 增速长期高于名义 GDP,2020–2022 实际利率深度为负 → **红**。
- **结论**:二维红一维黄,综合**红灯**——主权信用仍可滚动,但持有者面临"实际购买力缩水"风险。
- **结果**:达利欧自 2019 年起多次公开建议**配置 5–15% 黄金/通胀挂钩资产**,而非单纯美债。

### 案例 2:魏玛共和国 1923(书中货币体系演化案例)
- **问题**:为何持有马克国债与现金的市民被彻底清零?
- **方法论使用**:
  - 硬通货覆盖率:央行为保住汇率耗尽黄金/外汇储备 → **红**。
  - 现金流负担率:政府偿债挤占全部财政,赤字货币化启动 → **红**。
  - 实际利率:通胀月增 30%+,名义利率追不上 → **红**。
- **结论**:三维全红,**教科书级债务货币化崩塌**。
- **结果**:市民储蓄一文不值,回归"第一类货币"(实物黄金、外币、实物商品)——印证书中"现金和债券都会贬值甚至一文不值"。

## A2 — 触发场景 ★

**适用场景**(5+ 个具体触发):
1. 个人按揭 / 房贷决策:"月供占我家庭收入多少安全?"
2. 企业发债或杠杆收购:"公司能不能再借一轮?"
3. 主权债评估:"日本国债 / 美国国债 / 新兴市场债能买吗?"
4. 投资债券 ETF:"长期美债 ETF 还能持有吗?"
5. 杠杆扩张决策:"我要不要再加杠杆买第二套房 / 加仓股票?"
6. 公司贷款审查:"这家企业的应付账款能不能扛住?"
7. 地方政府融资 / 城投债评估。

**典型措辞**(5+ 个,触发词):
- "这笔债能不能借"
- "这个国家债务危险吗"
- "贷款安全吗 / 会不会爆"
- "我的房贷会不会爆"
- "我要不要加杠杆"
- "债券会不会贬值"
- "现金 vs 黄金,选哪个"

**与相邻 skill 的边界**:
- **vs `cycle-stage-detector`(周期阶段定位器)**:该 skill 看"我们现在处于信贷/债务大周期的哪一阶段",输出宏观定位;**本 skill 看具体债务主体的微观结构**。先定位阶段,再用三维评估具体一笔债。
- **vs `worst-case-decision-guard`(最坏情况决策守卫)**:该 skill 是**过程性决策框架**(预演最坏情况 → 设止损 → 反脆弱),本 skill 是**结构化风险量化工具**(三维数字打分)。前者是"该不该做",后者是"风险到底多大"。
- **vs 单一指标速判(如债务/GDP)**:达利欧强调"债务/GDP"忽略了现金流和硬通货维度,容易误判——日本债务/GDP > 250% 但仍可持续,正是因为低息 + 内部储蓄循环。

## E — 可执行步骤

1. **收集三比率输入数据**
   - 硬通货维度:央行外储、黄金储备、可快速变现的实物资产;外币计价债务总额。
   - 现金流维度:债务人过去 12 个月经营性现金流 / 家庭可支配收入;年到期本息。
   - 利息维度:同类债务名义到期收益率;近 24 个月 CPI、本币兑硬通货贬值率。

2. **计算三维评分**
   - 硬通货覆盖率 = 硬资产 / 外币债务
   - 现金流负担率 = 年还本付息 / 年现金流
   - 实际利率差 = 名义利率 − 预期本币贬值率

3. **逐维判断红/黄/绿**(按 I 段阈值)

4. **综合输出**
   - 全绿 → 绿灯:可承担,且有冗余。
   - 一黄 → 黄灯:可承担,但需限定规模和期限。
   - 一红 / 多黄 → 橙灯:结构性脆弱,必须缩短期限或要求抵押。
   - 两红及以上 → 红灯:**拒绝新增债务 / 主动去杠杆**。

5. **输出行动方案**(按主体类型):
   - 国家:增配黄金 / 外汇、缩短债务期限、降低对外计价负债占比。
   - 企业:砍资本开支、再融资拉长只到 2 年内、加股权融资。
   - 个人:控制房贷 / 消费贷 ≤ 现金流 30%,留 12 个月还本付息现金储备。
   - 投资者:本币贬值风险高时,减持长期债券 ETF,增持通胀挂钩债券或实物。

## B — 边界 ★

**反场景**(不要硬跑此 skill):
1. 短期交易决策(日内/周内择时)——三维是结构性指标,不反映动量。
2. 只有"债务/GDP"一个指标就想下结论——必失败,见案例 2(日本反例)。
3. 没有任何输入数据时硬跑——必须先收集现金流、储备、利率三项至少估算值。
4. 加密资产 / 稳定币估值——本框架基于主权法币体系,不适用。
5. 对冲基金杠杆 / 衍生品名义敞口——杠杆嵌套层级远超三维能覆盖。

**常见失败模式**:
1. **把"债务/GDP"等同于"债务/现金流"**——GDP 是流量但不是债务人可支配现金流;很多高负债国家 GDP 增速完全跟不上付息。
2. **忽略本币贬值预期的利息补偿**——只看名义收益率,不看实际利率差;魏玛 / 阿根廷式崩塌前夕,名义利率看似很高但跑不赢贬值。
3. **只看一维就下结论**——任何一维红都需独立警示,不能因另两维绿而忽视。
4. **把"硬通货"窄化为"黄金"**——硬通货指任何不受单一央行操控的价值贮藏:黄金 + 白银 + 大宗商品 + 强势外币储备 + 高流动性实物资产。

**作者盲点**:
- 达利欧框架基于**主权法币 + 银行信贷**体系,对**加密资产、稳定币、链上 DeFi 杠杆、影子银行嵌套**不适用。
- 对**小型开放经济体**的"硬通货维度"做了隐含假设(本币可自由兑换),对资本管制国家要打折使用。
- 对**政治冲击**(战争、革命、制裁)的外生黑天鹅,框架无法量化,需配合 `worst-case-decision-guard` 使用。

**邻近方法论**:
- **Altman Z-Score**:企业层面财务预警,只看企业财务报表;本框架覆盖国家/个人/企业,且纳入货币贬值预期。
- **CDS 利差分析**:市场实时定价,反映违约概率,但**不直接反映硬通货覆盖率和贬值预期**;两者互为补充。
- **债务/GDP 比率**(Reinhart–Rogoff):宏观杠杆指标,维度单一,被本框架视为"现金流维度"的弱代理。

## 数据源指引 (Data Sources)

本 skill 的三维比率(债务/硬通货、债务/现金流、利息 vs 贬值)必须用实时数据计算。调用时使用 WebFetch / WebSearch 从以下源获取最新数据:

- **World Bank IDS (外债数据)**: https://datatopics.worldbank.org/debt/ — 各国外债总额/短期外债/外储
- **IMF Data**: https://data.imf.org/ — 国际储备(国际投资头寸)
- **FRED 利率系列**: https://fred.stlouisfed.org/ — FEDFUNDS, DGS10, TIPS 实际利率
- **NBS 中国财政**: https://data.stats.gov.cn/ — 中国财政收入、地方政府债
- **财政部**: http://www.mof.gov.cn/ — 中国政府债务月度数据
- **BEA 美国**: https://www.bea.gov/ — 美国政府收入支出
- **BIS REER**: https://www.bis.org/statistics/ — 实际有效汇率(贬值信号)

**优先级**(按维度):
- 硬通货维度: World Bank IDS > IMF Data
- 现金流维度: NBS / 财政部 > BEA
- 利息维度: FRED > BIS REER

**查询提示**:
```
URL: https://datatopics.worldbank.org/debt/
prompt: 提取 [国家名] 最新外债总额、短长期外债、外汇储备,输出表格
```
