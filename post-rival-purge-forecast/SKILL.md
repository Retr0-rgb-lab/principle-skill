# post-rival-purge-forecast · 战后权力洗牌预测器

## R — 原文

**第 37 章 · 5 内部秩序和混乱大周期**

> 历史告诉我们,在打败了共同敌人的权力斗争之后,那些联手击败共同敌人的赢家通常会在内部争夺权力,而输家也会这样做,他们想要卷土重来。我把权力平衡机制的这一形态称为"清洗"。

**第 37 章 · 5 内部秩序和混乱大周期**

> 革命通常分为两部分:第一部分是推翻已有领导者和体制的斗争,第二部分是消除前领导者的忠诚分子的斗争以及胜者为争夺权力而进行的内斗。

**第 37 章 · 5 内部秩序和混乱大周期**

> 几乎所有内战都有一些外国势力参与,它们试图影响内战的结果,使其对自己有利。

**第 38 章 · 6 外部秩序和混乱大周期**

> 国际秩序与其说是遵循国际法,不如说是丛林法则。获取权力,尊重权力,并明智地运用权力。拥有权力是件好事,因为权力永远超越协议、规则和法律。

## I — 方法论骨架

把"共同敌人即将倒下"这一刻视为权力平衡的临界点,而不是胜利的终点。共敌一旦瓦解,维系联盟的粘合剂消失,所有原本被压抑的利益冲突会同时释放,洗牌以三种方向同时发生:

1. **战胜方联盟内斗**: 共敌消失 → 各成员按自己的利益重新排列,不再需要"分蛋糕"妥协。利益重叠越低,分裂速度越快;盟主实力越强,整合越稳,但整合期仍伴随次级盟友的离心。
2. **战败方内部清洗**: 原忠诚派被识别并清洗,幸存者按"战后如何生存"分裂为若干派系——投降派、流亡派、游击派、改革派。外部势力的代理人通常在此阶段介入并扶持某一派。
3. **外部势力介入**: 战胜方阵营之外的第三方会迅速填补真空,通过军援、情报、资本、外交承认影响战败方内部走向;战胜方联盟内部也可能被第三方渗透、分化。
4. **烈度评估**: 清洗烈度 = f(战败方崩溃速度、联盟利益重叠度、外部介入烈度)。烈度越大,新秩序重建越慢——达利欧明言"重建速度取决于内战/革命有多剧烈"。
5. **历史可类比性**: 同一阶段的洗牌模式高度可重复,因为人性、利益结构、外交博弈的逻辑跨时代稳定。

## A1 — 书中的应用

**案例 1 · 1945 二战结束 — 美苏从盟友变对手**

- **问题**: 轴心国崩溃后,英美苏三方如何从反法西斯同盟过渡到冷战格局?
- **方法论使用**: 共敌(德/日/意)消失 → 美苏按各自战后利益(势力范围、重建市场、意识形态)重新排列,利益重叠从"共同对敌"骤降至"分区占领、货币体系、原子能控制"全面冲突。英帝国因实力透支被动退入美国阵营。中国内部国共对抗,叠加美苏代理人介入。
- **结论**: 战胜方联盟内斗与战败方内部清洗同步发生,外部势力(美苏)深度介入他国内战,形成雅尔塔-冷战双层格局。
- **结果**: 1947 年杜鲁门主义、1949 年北约、1949 年中华人民共和国成立,印证"清洗不是胜利的终点,而是新周期的起点"。

**案例 2 · 拿破仑战争后维也纳体系(1815)**

- **问题**: 法国帝国崩溃后,欧洲如何避免新一轮霸权战争?
- **方法论使用**: 战胜方联盟(英、俄、普、奥)在击败拿破仑后立即面临"谁来主导欧洲"的争夺。英国担心俄普扩张、俄国想要波兰、普鲁士想要萨克森、奥地利想要意大利北部——利益高度重叠(均围绕"防止法国再起"之外的零和分配)。
- **结论**: 战胜方通过维也纳会议(1814-1815)+ 神圣同盟 + 四国同盟,建立"协商式霸权"机制,把内斗约束在会议桌上而非战场。
- **结果**: 欧洲获得约 100 年相对和平(1815-1914),印证"高烈度清洗可通过强制度压制为低烈度博弈",但代价是僵化秩序无法处理新兴力量(德/意统一)。

**案例 3 · 太平天国失败后(1864)与同期美国南北战争(1865)**

- **问题**: 重大内战结束后,战胜方联盟(清廷与列强、美国北方)如何处理战败方内部残余与外溢风险?
- **方法论使用**: 太平天国失败后,清廷依靠英法俄支持重建秩序,但湘军、淮军、地方督抚权力膨胀,埋下晚清"内轻外重"格局;同期美国南北战争后,联邦对南方实行重建方案,前邦联忠诚派被有限清洗,黑人获得法律地位但实际权利受限。
- **结论**: 两个案例都体现"外国势力介入 → 战胜方联盟重组 → 战败方内部派系分化",但烈度差异巨大——美国重建为高烈度短暂清洗,清廷为低烈度长期妥协,后果对应"新秩序重建速度"。
- **结果**: 美国在 1877 年结束重建后迅速工业化,清廷在 1895 年甲午战败,印证"清洗烈度决定重建速度"的达利欧命题。

## A2 — 触发场景

**适用场景(5+)**:
1. 重大政权即将崩溃的预测:用户问"X 政权崩溃后会发生什么""Y 领导人下台后谁会接班"。
2. 地缘战略部战后规划:智库、外交部、跨国企业做"后 X"情景推演。
3. 外交决策者:为战后秩序设计、和平协议谈判提供清洗风险评估。
4. 投资者:评估战争结束后资产价格走向、制裁解除节奏、跨境投资时机。
5. 历史案例比较分析:对照 1945、1815、1918、1991 等历史节点预测当代事件。
6. 学术研究:革命史、内战史、国际秩序变迁研究。

**典型措辞(5+)**:
- "共同敌人倒下后会怎样?"
- "联盟会不会散伙?"
- "打完胜仗会不会内斗?"
- "外国会不会趁机介入?"
- "清洗风险有多高?"
- "战后格局会变成什么样?"
- "谁是下一个 X 国?"

**与相邻 skill 的区分**:
- **vs `redline-mapper`**: `redline-mapper` 看当前各方利益分层(谁会为什么而战);本 skill 看共敌倒下后,利益结构动态崩塌后的洗牌过程。
- **vs `cycle-stage-detector`**: 阶段定位看国家在 100-300 年大周期哪一段;本 skill 看单一事件(共敌倒下)触发的短周期权力变迁。
- **vs `cycle-correlation-scenario`**: 那是宏观多情景推演(各国周期叠加);本 skill 是单一事件(共敌倒下)的三向洗牌路径。
- **vs `worst-case-decision-guard`**: 那是个体决策框架;本 skill 是结构化历史-地缘推演。

## E — 可执行步骤

1. **确认"共同敌人"是否接近失败**: 收集军事、经济、政治信号(领土丢失、领导层分裂、外援枯竭、内部哗变);若共敌仍有 1/3 以上实力与合法性,**停止推演**——本 skill 失效。
2. **列出战胜方联盟成员及利益结构**: 对每个盟友用 `redline-mapper` 的 X/Y/Z 框架做利益分层;重点关注 X 区域重叠度——重叠越低,内斗越快。
3. **列出战败方内部主要派系**: 投降派、流亡派、游击派、改革派、地方军阀;评估每个派系的资源、合法性、外部支持渠道。
4. **列出外部势力介入动机与渠道**: 战胜方阵营之外的第三方(大国、国际组织、邻国)按"想得到什么、能给什么"逐一分析;特别关注地理邻近大国。
5. **输出"洗牌路径图"**: 包含四块——
   - (a) **战胜方内斗矩阵**: 哪两方最先反目、利益冲突点、预计时间窗口(短期/中期/长期)
   - (b) **战败方清洗树**: 哪个派系被清洗、哪个派系被扶持、清洗烈度评分(低/中/高)
   - (c) **外部介入图**: 哪个外部势力支持哪个派系、介入形式(军援/外交/经济)、预计烈度
   - (d) **历史类比**: 从维也纳体系、1945 冷战、1991 苏联解体等案例中选 1-2 个,说明"像 + 不像"
6. **数据引用**:任何输出的数据点(数字、统计、事实)必须附带来源(URL 或数据库名 + 快照日期),无来源 → 不输出该数据点。例:`中国基尼系数 ≈ 0.47 (西南财大 CHFS 2018, https://...)`。

## B — 边界

**反场景(不要用)**:
- 共同敌人远未出现或远未接近失败——此时推演"战后"是空想。
- 双边冲突且无联盟——本 skill 专为"多对一"联盟结构设计。
- 用户对"清洗"机制的否认/无兴趣,只想听"和平稳定必胜"的乐观推演。
- 仅限国内问题、单一国家分析、无外部维度。

**常见失败模式**:
1. **把"和平协议已签"等同"洗牌结束"**: 协议往往是新冲突的起点,而非终点。
2. **忽视外部势力介入,只关注国内视角**: 达利欧明言"几乎所有内战都有外国势力参与",遗漏外部维度是严重盲点。
3. **战胜方联盟利益估计过于乐观,低估分裂速度**: 共敌消失后,利益冲突释放速度通常超出预期(1945 美苏只用了 24 个月就进入冷战)。
4. **战败方"民主化"过渡的简化假设**: 历史上战败方很少平稳民主化,更多经历军政府、军阀混战、长期游击战。
5. **用单一国家框架分析多极博弈**: 忽视第三方势力(尤其是地理邻近大国)的杠杆作用。
- **引用规则**:输出的所有数字、统计、事实性陈述必须附带可溯源出处(URL + 日期)。无法溯源的数据(民间说法、印象、听说)不得作为结论性陈述。

**作者盲点 / 时代局限**:
- 框架基于历史经验(尤其是 18-20 世纪),现代战争形式(网络战、AI 武器、瘫痪性打击)的"洗牌"速度可能比传统框架更快。
- 对"快速战胜"(闪电战)→ 战胜方联盟"继承性"的处理不到位:历史案例多为长期消耗战,当代技术可能让"共敌倒下→新冲突"间隔从数年压缩到数月。
- 框架以主权国家为主,对非国家行为体(跨国公司、恐怖组织、AI 平台)的角色估计不足。
- 过度强调"丛林法则",对国际制度、舆论压力、经济相互依赖的缓冲作用估计偏弱。

**邻近方法论的区分**:
- **和平研究(Peace Research, 如 Uppsala PRIO 冲突数据)**: 偏量化、统计驱动;本 skill 是结构化历史类比,输出方向而非概率。
- **国际关系新现实主义(Waltz/Gilpin 霸权稳定论)**: 关注国际体系结构;本 skill 关注共敌倒下后的"过渡时刻"动力学。
- **Cox 的历史唯物主义地缘框架**: 强调生产关系与阶级;本 skill 以利益结构与权力斗争为主轴,不引入阶级分析。
- **Schelling 焦点理论**: 研究协调点与可信承诺;本 skill 更宽,涵盖联盟重组与外部介入。

## 数据源指引 (Data Sources)

本 skill 评估战后权力洗牌需要历史数据 + 实时制度指标:

- **UCDP/PRIO 武装冲突数据**: https://ucdp.uu.se/ — 历史清洗烈度参考(年均死亡数、持续时间)
- **V-Dem 民主指数**: https://www.v-dem.net/ — 战后政权性质演变(从威权到民主的转型速度)
- **Polity V 历史档案**: https://www.systemicpeace.org/ — 政变、政权更迭事件库
- **World Bank GDP 历史**: https://data.worldbank.org/ — 战后重建速度(以重建期 GDP 增长率衡量)
- **IMF 债务可持续性分析**: https://www.imf.org/external/np/pubs/ft/dsa/ — 战后财政能力
- **SIPRI 军费**: https://www.sipri.org/databases/milex — 战胜方联盟军事实力变化
- **CIA World Factbook**: https://www.cia.gov/the-world-factbook/ — 政府类型变化历史

**优先级**:
1. UCDP (历史清洗案例库)
2. Polity V (政权更迭记录)
3. World Bank GDP (重建速度)

**查询提示**(对每个新政权):
```
URL: https://www.systemicpeace.org/polity/polity4.htm
prompt: 提取 [国家名] 过去 50 年政权更迭记录,标记与共敌消失事件的相关性
```
