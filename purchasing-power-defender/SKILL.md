# purchasing-power-defender · 购买力防御规划器

## R — 原文

**第 39 章 · 从大周期的角度思考投资**(投资定义)

> 投资的目的是以一种财富贮藏手段储存资金,以便今后将其转化成购买力。投资是一次性地支付未来的款项。因为你试图储存购买力,所以你必须考虑通胀因素。

**第 35 章 · 货币、信贷、债务和经济活动的大周期**(货币贬值 + 黄金论)

> 所有货币都会贬值甚至消亡。一旦出现这种情况,现金和债券(即获得货币的承诺)就会贬值甚至变得一文不值。…… 因此,黄金(其次是白银)既可以作为安全的交换媒介,也可以作为安全的财富贮藏手段。

**第 35 章 · 同章**(实物 vs 金融财富)

> 实物财富≠金融财富。实物财富具有内在价值。金融财富没有内在价值。央行印钞将导致货币贬值,使金融财富价值相对于实物财富价值下跌,直到金融资产的实际价值(经通胀调整后)低于实物资产的水平。

**第 51 章 · 审校后记**(世界末日组合 + 分散下注)

> 现在我有一个"世界末日"投资组合,我知道凭着它,我们就可以度过最糟糕的情况,这是我的基础。…… 如果我有一些颇具吸引力又**互不相关**(uncorrelated)的下注机会,我就可以在完全不影响收益的情况下把风险降低 80%。

## I — 方法论骨架

核心命题:**投资 = 储存购买力**,通胀是任何投资决策不可省略的输入。围绕"防止被淘汰出局",构建五层防御:

1. **底线防御层**——保留 6-18 个月生活费的现金 + 短期国债,目的是"不被淘汰出局",不是增值。
2. **实物资产层**——黄金(主) + 白银(次) + 大宗商品 / 不动产 / 抗通胀硬资产。逻辑:实物有内在价值,金融财富没有。
3. **分散下注层**——选择**真正互不相关**的资产(股、债、商品、房产、现金等价物),相同预期收益下风险可降低 80%。
4. **动态调整层**——"经常自问":利息能否覆盖货币贬值风险?若不能,触发再平衡。
5. **远离纯现金 / 长期债券层**(法币体系风险升高时)——以黄金 + 通胀挂钩债券(TIPS)替代。

五层并列,任一缺位都构成结构性脆弱。

## A1 — 书中的应用

**案例 1 · 达利欧"世界末日组合"(第 51 章自述)**

- **问题**:桥水面临极端情景(去美元化、滞胀、战争),常规组合扛不住。
- **方法**:底线(高流动性现金) → 实物(黄金为主) → 分散(TIPS + 大宗 + 抗通胀股) → 动态(季度重审"利息 vs 贬值")。
- **结论**:组合以"扛过最坏"为基准,而非追求最高收益。
- **结果**:成为达利欧个人与桥水"基础配置",2018 起多次公开复述。

**案例 2 · 2020 后美国家庭面对通胀的 5 层防御**

- **问题**:2020-2022 美国 M2 飙升约 40%,CPI 创 40 年新高,现金与长期国债实际收益为负。
- **方法**:底线增厚至 12-18 个月;实物层用黄金 ETF(5-15%)+ TIPS 替代纯长期国债;分散层选真正 uncorrelated 的股/债/商品;动态层季度复盘。
- **结论**:纯股票 + 现金的家庭购买力被通胀侵蚀;5 层防御下 2020-2024 实际购买力显著跑赢纯股票组合的最大回撤。
- **结果**:印证达利欧"实物有内在价值,金融财富没有"。

**案例 3 · 1923 魏玛 / 1970s 美国滞胀**

- **问题**:马克 / 美元现金与债券购买力数年内清零或腰斩。
- **方法**:对照 5 层——底线失效、长期债券成为负债,仅实物层(黄金 + 大宗 + 不动产)幸存。
- **结论**:5 层任一缺失都意味着法币崩塌时出局;实物层是不可替代的"逃生通道"。
- **结果**:5 层防御在极端货币体系崩塌中验证。

## A2 — 触发场景 ★

**适用场景**(6+):

1. 通胀起来后焦虑"现金缩水",需分层防御方案。
2. 年轻家庭长期规划,首次建立跨周期配置。
3. 临近退休需再平衡,从进攻转防御。
4. 跨国资产分配(移民 / 海外置业 / 多币种生活)。
5. 机构(基金会、家族办公室)资产抗通胀框架。
6. 危机后(疫情、风暴、货币贬值)重建个人配置。

**典型措辞**(6+):

- "钱缩水怎么办?" / "该买黄金吗?"
- "怎么抗通胀?"
- "达利欧世界末日组合是什么?"
- "怎么不靠政府保护钱财?"
- "该买实物资产吗?"
- "现金 vs 黄金,选哪个?"

**与相邻 skill 的区分**:

- **vs `currency-crisis-scanner`**:本 skill 是**防御方案**(How to defend),后者是**危机预警**(Is it coming);先预警,再防御。
- **vs `debt-risk-triple-ratio`**:本 skill 看**资产端**,后者看**负债端**;两者互为镜像。
- **vs `worst-case-decision-guard`**:本 skill 是**资产框架**,后者是**决策框架**(过程性);本 skill 是"防淘汰"四步法在资产上的落地。

## E — 可执行步骤

1. **评估货币风险处境**:收集本币 5 年 CPI、外储覆盖率、央行独立性评分;可用 `currency-crisis-scanner` 输出作输入。
2. **确定 5 层权重**:低风险档(底线 30% / 实物 10% / 分散 50%);高风险档(底线 40% / 实物 25% / 分散 25%,现金用 TIPS 替代)。
3. **列出每层具体工具 + 比率**:底线 → 货币基金 + 3 月内国债;实物 → 实物金 > 黄金 ETF > 白银;分散 → 股票指数 + 商品指数 + REITs + 短久期国债。
4. **设置复盘节奏**:季度复盘"利息 vs 贬值"差;年度再平衡(偏离 ±5% 触发)。
5. **输出"防御方案 + 触发调整条件"**:若本币贬值突破阈值 / CPI > 5% / 外储 < 30%,触发防御升级(如实物层从 10% 提至 20%)。

## B — 边界 ★

**反场景**:

1. **短期交易 / 做波段**——5 层是结构性配置,不适合月内调仓。
2. **追求"最高收益"而非"防御"**——目标冲突,需先对齐"防御"目标。
3. **无风险处境信息就硬跑**——缺乏 CPI / 外储输入,无法决定权重调档。

**常见失败模式**:

1. 把"分散"等同"少买一类资产",未做到真正 uncorrelated(同业 10 支股票不是分散)。
2. 买黄金不配白银,过度集中;实物层应金/银/大宗多元。
3. 忽视动态调整,买入躺平;不复盘"利息 vs 贬值"。
4. 把"实物"窄化为"黄金";不动产、抗通胀股、大宗同样属实物层。
5. 用长期债券替代现金层;长久期债在加息 + 通胀下双重亏。

**作者盲点**:

- 5 层防御隐含**高净值 / 全球可配置**前提;中低收入家庭需简化(底线 + 实物 + 分散三档)。
- 达利欧对加密资产持保留,本 skill 受此约束,不把 BTC/ETH 列为实物层默认工具。
- "真正 uncorrelated"在低相关性市场(2020s 股债同跌)下失效,需配合期权 / 波动率补充。

**邻近方法论**:

- **Markowitz 现代投资组合理论**:数学最优化,假设收益率正态;本 skill 是经验性防御框架,对尾部事件不依赖分布假设。
- **Harry Browne"永久投资组合"**:固定 4 档(股 25% / 长债 25% / 黄金 25% / 现金 25%);本 skill 是 5 层 + 动态调整,可按风险调档。
- **耶鲁模式(David Swensen)**:偏机构 + 长期股权 + 另类资产;本 skill 偏个人/家庭防御。
