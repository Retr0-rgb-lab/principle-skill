# redline-mapper · 地缘底线映射器

## R — 原文

**第 38 章 · 外部秩序和混乱大周期**
> 国际秩序与其说是遵循国际法，不如说是丛林法则。

**第 38 章 · 外部秩序和混乱大周期**
> 重要的是，要知道哪些是对手最重要的利益，哪些是其不介意的东西，特别是要了解对手会为什么而战，不会为什么而战。

**第 34 章 · 决定因素**
> 要想进一步降低发生冲突的风险，双方就需要交换利益，建立相互依赖的关系，使自身不能承受失去这种关系的后果。

**第 37 章 · 内部秩序和混乱大周期**
> 几乎所有内战都有一些外国势力参与，它们试图影响内战的结果，使其对自己有利。

## I — 方法论骨架

把地缘冲突先画成“利益分层图”，而不是直接判断谁对谁错。关键是识别对手真正的核心利益，防止把可交易事项误判成战争理由，或把红线当成讨价还价。

1. **X 核心利益**：会为之而战，基本不可交易，只能避让、承认或威慑。
2. **Y 重要但可交易**：优先级高，但可用时间、形式、范围、补偿交换。
3. **Z 不介意**：低成本让步，可作为筹码、面子或缓冲安排。
4. **重叠分析**：双方 Y-Z 相邻处是谈判空间；双方 X 重叠处是红线冲突。
5. **力量测试风险**：X-X 重叠且缺少裁决机制时，争端容易转为制裁、代理冲突或军事测试。
6. **缓冲评估**：检查贸易、金融、能源、技术、安全等相互依赖，尤其看是否“不能承受失去”，以及依赖是否非对称。

## A1 — 书中的应用

**案例 1 · 美苏冷战/古巴导弹危机**
- **问题**：美国把近岸核导弹视为本土安全红线，苏联也要保住战略威慑与盟友信誉。
- **方法论使用**：美国 X 是移除古巴进攻性导弹；苏联 X 是不被羞辱性击退。土耳其导弹、公开措辞、撤出节奏属于 Y/Z。
- **结论**：避免 X-X 硬撞，改用 Y/Z 交换与面子安排。
- **结果**：危机降温，说明红线清楚时，交易空间反而更容易被找到。

**案例 2 · 中美台海**
- **问题**：中国的主权统一、美国的地区秩序与信誉、台湾的安全自治彼此交叠。
- **方法论使用**：把“独立/武力改变现状/外部军事介入”等划入各方 X；经贸、沟通机制、访问规格、时间表多属 Y/Z。
- **结论**：X 区域高度接近，真正可谈的是形式、节奏、沟通与危机管控。
- **结果**：若公开逼迫对方否认 X，力量测试风险上升；若保留模糊与依赖，缓冲增强。

**案例 3 · 北约—俄罗斯/乌克兰 2014-2022**
- **问题**：俄罗斯把周边安全缓冲视为核心，乌克兰把主权领土视为核心，北约强调开放门与欧洲安全秩序。
- **方法论使用**：三方 X 重叠过多，部署限制、时间表、核查、经济安排等 Y 未能形成足够缓冲。
- **结论**：当安全缓冲与主权边界都被放入 X，谈判空间急剧收窄。
- **结果**：依赖关系不足以承受政治压力，争端转入力量测试。

## A2 — 触发场景

**适用场景(5+)**：
1. 外交谈判前，整理双方红线与可让步项。
2. 企业做地缘风险管理，判断制裁、禁运、撤资风险。
3. 记者写国际报道背景，避免只复述公开口号。
4. 跨国供应链评估，找依赖链上的缓冲器与断点。
5. 公众政策讨论，区分“立场表演”和“核心利益”。
6. 智库/投资者评估台海、乌克兰、朝鲜半岛等冲突升级路径。

**典型措辞(5+)**：
- “中美底线是啥？”
- “为啥而战，不为啥而战？”
- “台海有妥协空间吗？”
- “他们的红线是什么？”
- “双方可交换的筹码是什么？”
- “哪些只是面子，哪些真会打？”

**与相邻 skill 的区分**：
- **vs `cycle-stage-detector`**：阶段定位看国家处在大周期哪一段；本 skill 看两个或多个竞争方的利益对位。
- **vs `worst-case-decision-guard`**：本 skill 是客观利益地图；后者是用户在不确定性下的主观决策与防淘汰框架。

## E — 可执行步骤

1. **收集各方立场文件**：官方声明、条约、领导人讲话、制裁清单、军事部署、贸易依赖数据。
2. **把每方诉求按 X/Y/Z 分层**：X=会为之而战；Y=重要但可交易；Z=不介意或低成本让步。
3. **找重叠冲突区**：标出双方 X 都重叠的位置；再找 Y-Z、Y-Y 的交换空间。
4. **评估相互依赖程度**：贸易、能源、金融、技术、侨民、安全承诺；判断是否存在“不能承受失去”的缓冲，以及哪方更依赖。
5. **输出“利益矩阵+红线圈定”**：列 X/Y/Z 表、可谈判空间、红线风险、力量测试概率、降低误判建议。

## B — 边界

**反场景(不要用)**：
- 纯历史回顾，只问发生过什么。
- 单一国家无明确对手。
- 对手是恐怖组织等非国家行为体，利益层级不稳定。
- 用户追求“最终判断/站队”，不接受利益图。

**常见失败模式**：
1. 把战术策略混淆为国家层核心利益，例如把一次军演直接等同开战红线。
2. 忽略非对称依赖：一方“不能承受失去”，另一方只是“不方便失去”。
3. 把公开立场等同核心利益，忘记公开话术可能是谈判锚点或国内政治姿态。

**作者盲点**：达利欧主要采用主权国家框架；企业—国家混合博弈需扩展到监管、资本、技术标准。对文化、宗教、意识形态驱动的非利益冲突，也不能只用交易逻辑解释。

**邻近方法论**：
- **博弈论**：追求数学最优策略；本 skill 先做利益分类与误判控制。
- **零和 vs 正和**：判断收益结构；本 skill 进一步标出哪些收益可交换、哪些不可交换。
- **Schelling 焦点理论**：研究协调点与可信承诺；本 skill 更侧重红线识别与缓冲依赖。
