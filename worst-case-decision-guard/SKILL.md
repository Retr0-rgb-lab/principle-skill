# worst-case-decision-guard · 最坏情况决策保护器

## R — 原文

**第 51 章 · 审校后记**(四步法原话集中)

> 我所获得的一切成就,主要不是由于我知道什么,而是由于我知道如何应对我所不知道的东西。

> 了解所有的可能性,考虑最坏的情况,然后想办法消除无法忍受的情况。首先要确定和消除无法忍受的最坏情况。**在生活或市场的博弈中,最重要的是不要被淘汰出局**。

> 我经常计算,假如没有任何进项,我和我的家庭可以撑几周、几个月或者几年。现在我有一个"世界末日"投资组合,凭着它就可以度过最糟糕的情况,这是我的基础。

> 如果我有一些颇具吸引力又**互不相关**的下注机会,我就可以在完全不影响收益的情况下把**风险降低 80%**。

> **与最聪明的人反复沟通**。我紧跟我能找到的最聪明的人,这样我就能测试我的想法,并向他们学习。

## I — 方法论骨架

四步递进逻辑链:

1. **穷举(认知)**:不盯最可能结果,把全谱系列出——从最好到最坏,含尾部事件。
2. **红线(隔离)**:把"被淘汰出局"级与"可承受损失"分开,单独标记。
3. **分散(防御)**:针对红线逐一设计对冲——留 6-18 个月流动性、配互不相关资产、保留退出路径。
4. **压力测试(校验)**:把方案交给最聪明的人,尤其会反对你的,让质疑挖出盲区。

链条本质:用系统方法兜底"我不知道"的部分,不为某个特定结果下注。

## A1 — 书中的应用

**案例 1 · 1982 年美国债务危机预测**
- **问题**:达利欧判断美国将进入债务危机,但市场主流持续看涨。
- **方法**:做 200 年债务复盘 → 穷举结果 → 标"美元体系崩塌"为出局级红线 → 留对冲仓位 → 与反对者压力测试。
- **结果**:1981-82 年预言应验,桥水因对冲头寸避免被淘汰,反而吸引后续资金。
- **结论**:与共识相反的判断,没有"防淘汰"框架根本不敢持仓。框架让人敢为少数派观点负责。

**案例 2 · 全职创业决策**
- **问题**:稳定工作者想辞职全职创业,担心失败后无法翻身。
- **方法**:列 8 个可能结果(翻倍/持平/腰斩/破产等) → 红线定为"房贷断供""家庭 1 年保障不足" → 防御:留 6 个月流动资金、不卖唯一住房、约好兼职退路 → 交由"创业失败过的朋友"质疑。
- **结果**:创业推进同时保留安全边际;项目两年后失败仍可恢复,不被淘汰。
- **结论**:防淘汰让人更敢尝试,而非更保守——因为出局风险已隔离。

## A2 — 触发场景

**适用场景(5+)**:
1. 全职 / 辞职创业。
2. 辞职转行或换城市。
3. 重大投资决策(股票、房产、杠杆)。
4. 跨国家庭迁移。
5. 买房买车的高杠杆决策。
6. 婚姻 / 生育等家庭重大决定。

**典型措辞(5+)**:
- "风险太大我不敢决定。"
- "最坏会怎样?" / "万一错了怎么办?"
- "我该怎么办?"
- "这个决策的底线是什么?"
- "我能承受失败吗?"
- "太多不确定性。"

**与相邻 skill 的区分**:
- **`cycle-stage-detector`**:宏观坐标定位(身处周期哪一阶段);本 skill 是个人 / 组织决策框架,以阶段为输入,产出决策清单。
- **`debt-risk-triple-ratio`**:结构化风险量化(三个比率),衡量已存在风险;本 skill 是过程方法,用于设计新决策。

## E — 可执行步骤

1. **明确决策问题**:一句话写清"我要决定什么"。
2. **穷举 5-10 个可能结果**:覆盖最好 / 中等 / 最坏,含尾部事件。
3. **标记"被淘汰级"红线**:列出 1-3 个绝不能发生的结果(财务出局、职业归零、健康崩塌)。
4. **设计防淘汰方案**:每条红线至少配一个对冲(流动性 / 互不相关下注 / 退出路径)。
5. **列出压力测试的"聪明人"**:找 2-3 个会质疑你的人,预约沟通。

## B — 边界

**反场景(不要用)**:
- 日常琐碎选择(午餐、周末去哪)。
- 已确定决策只需执行。
- 零风险或完全可逆场景。
- 用户不愿参与推演,只要结论。

**常见失败模式**:
1. **漏红线**——只想 20% 的小损失,忽略 1% 的出局事件。
2. **把"分散"当"少买"**——同业股票 10 支不叫分散;真正分散是"互不相关"。
3. **压力测试流于形式**——找支持者喝彩,不是找反对者质疑。
4. **把"最坏"想成"小损"**——而非"低概率但出局级"。

**作者盲点**:
- 个体/小团队框架套大型组织时退出成本不同,需调整。
- 高净值的红线标准(数千万美元级)不适用于普通家庭,需按个人尺度重校。

**邻近方法论**:
- **决策树分析**:量化分支概率,易陷"精确的错误";本 skill 重红线隔离,对精度要求低。
- **Black-Scholes 期权对冲**:假设市场有效、波动可估;本 skill 是过程框架,不依赖特定工具。
- **Prospect Theory**:描述人在损/益前的非理性(描述性);本 skill 是规范方法(规范性)。
